
ElectroStorePOS - Sistema POS para tienda de componentes electrónicos

1. Instalación:
   - Descomprime el ZIP
   - Ejecuta: composer install
   - Copia el archivo .env.example a .env y configura la base de datos
   - Ejecuta: php artisan migrate --seed
   - Inicia el servidor: php artisan serve

2. Usuarios de prueba:
   - Admin: admin@electrostore.com / admin123
   - Cajero: cajero@electrostore.com / cajero123
   - Proveedor: proveedor@electrostore.com / proveedor123
   - Cotizador: cotizador@electrostore.com / cotizador123

3. Características:
   - Ventas con recibos PDF + QR
   - Gestión de stock y alertas visuales
   - Roles diferenciados
   - Modo claro/oscuro
   - Diseño tipo tienda

Gracias por usar ElectroStorePOS!
